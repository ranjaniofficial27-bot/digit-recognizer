# Handwritten Digit Recognizer 
A deep learning project using CNNs to classify handwritten digits from the MNIST dataset with ~99% accuracy. Includes real-time digit drawing with Tkinter.

 Overview
A deep learning project using CNNs to classify handwritten digits from the MNIST dataset with ~99% accuracy. Includes real-time digit drawing with Tkinter.

Skills Learned
- Image preprocessing
- CNN architecture design
- Model training & evaluation (TensorFlow/Keras)
- Real-time prediction with Tkinter GUI

Tech Stack
- Python
- TensorFlow/Keras
- Tkinter
- Matplotlib / Seaborn

 Features
- Train CNN on MNIST dataset
- Confusion matrix & accuracy graphs
- Real-time digit drawing (Tkinter)

Results
- Test Accuracy: ~99%
- Confusion Matrix shows minimal misclassifications

How to Run
```bash
pip install -r requirements.txt
python train_model.py
